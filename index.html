Что делает Полностью отключает Microsoft Defender.
Отключение 3 этапа перезагрузки. Обходит Tamper Protection через Safe Mode и захват владения реестром. 120+ настроек реестра, службы/драйверы Defender, SmartScreen, DISM, Windows Update.
Статус Аудит служб Defender + Windows Update, драйверов, процессов, реестра, WMI, SmartScreen.
Стек PowerShell + BAT-обёртка, логи, автозапуск от администратора.
